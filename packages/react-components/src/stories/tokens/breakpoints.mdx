# Storybook Design Tokens

Display design token documentation generated from your stylesheets and icon files. Preview design token changes in the browser. Add your design tokens to your Storybook Docs pages using the custom Doc Blocks.

**Breakpoints:**
* `xs`: 0px (Applies globally, `width >= 0`)
* `sm`: 0px (Applies globally, `width >= 0`)
* `md`: 960px (`min-width: 960px`)
* `lg`: 1264px (`min-width: 1264px`)
* `xl`: 1904px (`min-width: 1904px`)

---

## Visibility Classes (`hide-` and `show-`)

Control the display of elements at different breakpoints.

| Class Name            | Description           | Usage Example (HTML)               | Actual CSS Applied              |
| :-------------------- | :-------------------- | :--------------------------------- | :------------------------------ |
| `.hide-xs`            | Hides element on extra-small screens and up | `<div class="hide-xs">...</div>` | `display: none;`                |
| `.hide-sm`            | Hides element on small screens and up | `<div class="hide-sm">...</div>` | `display: none;`                |
| `.hide-md`            | Hides element on medium screens and up | `<div class="hide-md">...</div>` | `@media (min-width: 960px) { display: none; }` |
| `.hide-lg`            | Hides element on large screens and up | `<div class="hide-lg">...</div>` | `@media (min-width: 1264px) { display: none; }` |
| `.hide-xl`            | Hides element on extra-large screens and up | `<div class="hide-xl">...</div>` | `@media (min-width: 1904px) { display: none; }` |
| `.show-sm`            | Displays element as block on small screens and up | `<div class="show-sm">...</div>` | `display: block;`               |
| `.show-md`            | Displays element as block on medium screens and up | `<div class="show-md">...</div>` | `@media (min-width: 960px) { display: block; }` |
| `.show-lg`            | Displays element as block on large screens and up | `<div class="show-lg">...</div>` | `@media (min-width: 1264px) { display: block; }` |
| `.show-xl`            | Displays element as block on extra-large screens and up | `<div class="show-xl">...</div>` | `@media (min-width: 1904px) { display: block; }` |

**Note**: `show-xs` is not generated by the provided SCSS. `show-sm` applies globally due to its breakpoint definition (`width >= 0`).

## Column Sizing (`col-`)

Defines the width of a grid column based on a 12-column system. `N` ranges from 1 to 12.

| Class Name              | Description                 | Usage Example (HTML)                    | Actual CSS Applied (Example for N=6) |
| :---------------------- | :-------------------------- | :-------------------------------------- | :----------------------------------- |
| `.col-xs-N`             | Sets column width for extra-small screens and up | `<div class="col-xs-6">...</div>`    | `flex-basis: 50%; max-width: 50%;`  |
| `.col-sm-N`             | Sets column width for small screens and up | `<div class="col-sm-4">...</div>`    | `flex-basis: 33.333%; max-width: 33.333%;` |
| `.col-md-N`             | Sets column width for medium screens and up | `<div class="col-md-6">...</div>`    | `@media (min-width: 960px) { flex-basis: 50%; max-width: 50%; }` |
| `.col-lg-N`             | Sets column width for large screens and up | `<div class="col-lg-3">...</div>`    | `@media (min-width: 1264px) { flex-basis: 25%; max-width: 25%; }` |
| `.col-xl-N`             | Sets column width for extra-large screens and up | `<div class="col-xl-12">...</div>` | `@media (min-width: 1904px) { flex-basis: 100%; max-width: 100%; }` |

**CSS Calculation Example:** For `.col-#{$bp}-N`, the width is `calc(N / 12) * 100%`.
* `.col-X-1` = `8.333%`
* `.col-X-3` = `25%`
* `.col-X-6` = `50%`
* `.col-X-12` = `100%`

## Order Classes (`order-`)

Controls the visual order of flex items within their container. `N` ranges from 1 to 12.

| Class Name              | Description                 | Usage Example (HTML)                    | Actual CSS Applied (Example for N=2) |
| :---------------------- | :-------------------------- | :-------------------------------------- | :----------------------------------- |
| `.order-xs-N`           | Sets order for extra-small screens and up | `<div class="order-xs-2">...</div>`  | `order: 2;`                         |
| `.order-sm-N`           | Sets order for small screens and up | `<div class="order-sm-1">...</div>`  | `order: 1;`                         |
| `.order-md-N`           | Sets order for medium screens and up | `<div class="order-md-3">...</div>`  | `@media (min-width: 960px) { order: 3; }` |
| `.order-lg-N`           | Sets order for large screens and up | `<div class="order-lg-4">...</div>`  | `@media (min-width: 1264px) { order: 4; }` |
| `.order-xl-N`           | Sets order for extra-large screens and up | `<div class="order-xl-5">...</div>` | `@media (min-width: 1904px) { order: 5; }` |

## Offset Classes (`offset-`)

Adds left margin to an element, effectively "offsetting" it by a specified number of columns. `N` ranges from 0 to 12.

| Class Name              | Description                 | Usage Example (HTML)                    | Actual CSS Applied (Example for N=6) |
| :---------------------- | :-------------------------- | :-------------------------------------- | :----------------------------------- |
| `.offset-xs-N`          | Adds offset for extra-small screens and up | `<div class="offset-xs-6">...</div>` | `margin-left: 50%;`                 |
| `.offset-sm-N`          | Adds offset for small screens and up | `<div class="offset-sm-2">...</div>` | `margin-left: 16.667%;`             |
| `.offset-md-N`          | Adds offset for medium screens and up | `<div class="offset-md-4">...</div>` | `@media (min-width: 960px) { margin-left: 33.333%; }` |
| `.offset-lg-N`          | Adds offset for large screens and up | `<div class="offset-lg-1">...</div>` | `@media (min-width: 1264px) { margin-left: 8.333%; }` |
| `.offset-xl-N`          | Adds offset for extra-large screens and up | `<div class="offset-xl-0">...</div>` | `@media (min-width: 1904px) { margin-left: 0; }` |

**CSS Calculation Example:** For `.offset-#{$bp}-N`, the margin is `calc(N / 12) * 100%`.
* `.offset-X-0` = `0%`
* `.offset-X-1` = `8.333%`
* `.offset-X-6` = `50%`
* `.offset-X-12` = `100%`
